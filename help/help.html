    JS87EMU is an emulator of a rare Soviet note book Elektronika MK87.

    The task of the emulator is to emulate the device as accurately as possible, taking into account its technical features. This emulator introduces for the first time the soft switch emulation and the Elektronika 51 clock simulation. It also features a PDP-11 (notebook processor architecture) debugger.

    The help describes the features of working with the emulator and its subsystems. If you have not previously had experience working with MK 87 (real or through another emulator), it is recommended that you first read the original instruction manual, where you can find basic information about the note book and the principles of working with it. Unfortunately, there is only a Russian version of the manual, which makes it difficult for foreign users to understand. The translation of images does not provide 100% accuracy, so for simplicity, you can also study the manual of the "prototype" device - Casio PF-3000. While the MK87 lacks some features and there are some differences in the operation of the system, much of the MK87 documentation is clearly borrowed from the PF-3000 documentation.
