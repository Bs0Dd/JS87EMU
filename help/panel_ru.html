Панель опций


    Панель опций является инструментом для настройки и управления эмулятором, позволяя модифицировать базовые параметры, влияющие на поведение виртуальной записной книжки.

    Панель состоит из 4 секций со сгруппированными по общему признаку параметрами.


Секция основных функций


    На данной панели находятся основные кнопки для вызова некоторых функций записной книжки.
    Кнопка "Init memory" соответствует нажатию комбинации клавиш "СИ"+"С"(не буква)+"СК", запуская процесс сброса записной книжки к начальному состоянию. Необходим, если нужно удалить все записи из книжки или инициализировать чистый образ ОЗУ. При нажатии, на экране на несколько секунд появляется надпись "** СБРОС! **".
    Кнопка "Show key" соответствует нажатию недокументированной комбинации клавиш "СИ"+"СК"+"4", при которой записная книжка показывает секретное слово, заданное пользователем для защищенного сектора информации. Если слово не задано, на экране появится надпись "СЕКРЕТА НЕТ". Для выхода нажать клавишу "С"(не буква).
    Кнопка "Easter egg" соответствует нажатию недокументированной комбинации клавиш "С"(не буква)+"СК"+"К", которая активирует "пасхальное яйцо" (подобное есть в МК 85 по команде WHO), выводящее на экран строку "ПРОГРАММУ РАЗРАБОТАЛ ПОДОРОВ А.Н.". Для выхода нажать клавишу "С"(не буква).
    Кнопка "Unknown function" соответствует нажатию недокументированной комбинации клавиш "С"(не буква)+"СК"+"Н", которая определена в прошивке, но ее назначение на данный момент неизвестно. Возможно для этой комбинации планировалось какое-то действие, но в итоге его не реализовали, оставив только зарезервированную комбинацию в коде.
    Кнопка "Reset ROM" позволяет сбросить состояние ПЗУ на заводское, если содержимое менялось с помощью лайт-ассемблера.
    Кнопка "Reset Clock (11:11)" соответствует нажатию двух кнопок на модуле часов, что приводит их к сбросу. Согласно инструкции, при сбросе на часах устанавливаются следующие показания: 11 часов, 11 минут, 11 секунд, 11 день, 11 месяц, 11 год, день недели - понедельник.
    Кнопка "Set real time" устанавливает на часах текущее системное время.


Секция управления


    Секция служит для управления работой эмулятора и переключения общих функций. Кнопки "Pause" и "Restart", как следует из названия, служат для приостановки и перезапуска эмулятора. Параметр "Vibro (keys)" включает или выключает вибрацию устройства (если таковое способно вибрировать, что, как правило, касается портативных устройств) при нажатии на виртуальную клавиатуру записной книжки. На устройствах без возможности вибрации пункт не выполняет никакой функции.


Секция загрузки и сохранения данных


    В этой секции можно загружать или сохранять ОЗУ виртуальной записной книжки.
    В поле "RAM file" отображается название загруженного файла ОЗУ памяти. Надпись "internal" (внутренний) показывается, если ОЗУ образ создан кнопкой "Clear RAM". При создании чистого ОЗУ кнопкой "Clear RAM" необходимо произвести инициализацию памяти, зажав комбинацию клавиш "СИ"+"С"(не буква)+"СК" или сымитировав их нажатие кнопкой "Init memory" в секции основных функций. На экране на несколько секунд загорится надпись "** СБРОС! **".
    Чтобы загрузить ОЗУ из бинарного файла, достаточно выбрать файл (кнопкой или через Drag'n'Drop) и нажать кнопку "Load RAM from file". При этом файл должен быть равен 2КБ. При несоблюдении условия образ будет автоматически приведен к 2КБ.
    ОЗУ также можно загрузить по сетевой ссылке с помощью URL аргумента ram. Например: http://example.com/emu.html?ram=http://example.com/ram.bin. Если в эмуляторе имеется сохраненное ОЗУ, перед загрузкой пользователю нужно подтвердить, что он согласен загрузить ОЗУ по сетевой ссылке, поскольку это приведет к потере сохраненного ОЗУ. Используя параметр fload=force можно отключить подтверждение, но добавлять параметр в публичные ссылки категорически не рекомендуется!
    Получить бинарный файл образа ОЗУ можно кнопкой "Save RAM", при этом браузером загрузится файл "mk87_ram.bin".


Секция дополнительных функций


    В секции находятся кнопки для открытия утилит и информационных окон, а также отладочные параметры.
    Кнопка "Keyboard layout" открывает окно со схемой раскладки компьютерной клавиатуры, адаптированной для работы с виртуальной записной книжкой. Кнопка "Debugger" открывает, соответственно, панель Дебаггера. Кнопка "Help" открывает окно с данной справкой. А кнопка About - окно с краткими сведениями о программе и списком изменений (Changelog).
    Параметр "Show debug messages" разрешает показ отладочных сообщений в консоли (через функцию console.log).
    Параметр "Disable soft poff" отключает поддержку присутствующего в процессоре программного выключателя, позволяющего прошивке выключать устройство. В МК 87 программное выключение происходит по программно-реализованному таймеру, считающему время до выключения. Поскольку прошивка никогда не останавливает тактовый генератор процессора (в отличии от МК 85), постоянно работает программа опроса клавиатуры, которая, при отсутствии нажатой клавиши, уменьшает счетчик таймера (при нажатии клавиши таймер сбрасывается). По истечении примерно 3.5 минут таймер иссякнет и устройство автоматически выключится в результате длительного бездействия.
