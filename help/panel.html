Options panel


    Options panel is a tool for configuring and managing the emulator, allowing you to modify basic parameters that affect the behavior of the virtual note book.

    The panel consists of 4 sections with parameters grouped by common feature.


Main functions section


    This panel contains the main buttons for calling some functions of the notebook.
    The "Init memory" button corresponds to pressing the key combination "СИ"+"С"(not a letter)+"СК", starting the process of resetting the note book to the initial state. Required if you need to delete all entries from the note book or initialize a clean RAM image. When pressed, the message "** СБРОС! **" (Reset) appears on the screen for several seconds.
    The "Show key" button corresponds to pressing the undocumented key combination "СИ"+"СК"+"4", in which the notebook shows the secret word set by the user for the protected information sector. If the word is not set, the message "СЕКРЕТА НЕТ" (No Secret) will appear on the screen. To exit, press the "С" key (not a letter).
    The "Easter egg" button corresponds to pressing the undocumented key combination "С"(not a letter)+"СК"+"К" which activates the "Easter egg" (there is a similar one in MK 85 by the WHO command), which displays the line "ПРОГРАММУ РАЗРАБОТАЛ ПОДОРОВ А.Н." (The program was developed by Podorov A.N.). To exit, press the "С" key (not a letter).
    The "Unknown function" button corresponds to pressing the undocumented key combination "С"(not a letter)+"СК"+"Н", which is defined in the firmware, but its purpose is currently unknown. Perhaps, some action was planned for this combination, but in the end it was not implemented, leaving only a reserved combination in the code.
    The "Reset ROM" button allows you to reset the state of the ROM to the factory default if the contents were changed using lite-assembler.
    The "Reset Clock (11:11)" button corresponds to pressing two buttons on the clock module, which causes it to reset. According to the instructions, when resetting the clock the following values are set: 11 hours, 11 minutes, 11 seconds, 11 day, 11 month, 11 year, day of the week - Monday.
    The "Set real time" button sets the current system time on the clock.


Control section


    The section is used to control the emulator and switch general functions. The "Pause" and "Restart" buttons, as the name suggests, are used to pause and restart the emulator. The "Vibro (keys)" parameter enables or disables device vibration (if it can vibrate, which usually applies to portable devices) when pressing the virtual keyboard of the address book. On devices without vibration capability, the item does not perform any function.


Data loading and saving section


    In this section you can load or save the RAM of the virtual note book.
    The "RAM file" field displays the name of the loaded RAM file. The "internal" caption is displayed if the RAM image was created using the "Clear RAM" button. When creating clean RAM with the "Clear RAM" button, you need to initialize the memory by holding down the "СИ" + "С" (not a letter) + "СК" key combination or simulating pressing them with the "Init memory" button in the main functions section. The "** СБРОС! **" (Reset) message will light up on the screen for several seconds.
    To load RAM from a binary file, just select the file (using the button or via Drag'n'Drop) and press the "Load RAM from file" button. The file must be 2KB. If the condition is not met, the image will be automatically reduced to 2KB.
    RAM can also be loaded from a network link using the URL argument ram. For example: http://example.com/emu.html?ram=http://example.com/ram.bin. If the emulator has saved RAM, before loading, the user needs to confirm that he agrees to load the RAM from network link, as this will result in loss of the saved RAM. Using the fload=force parameter you can disable confirmation, but adding the parameter to public links is strictly not recommended!
    You can get the binary file of the RAM image by clicking the "Save RAM" button, and the browser will download the "mk87_ram.bin" file.


Additional functions section


    The section contains buttons for opening utilities and information windows, as well as debugging parameters.
    The "Keyboard layout" button opens a window with a layout scheme of a computer keyboard, adapted for working with a virtual note book. The "Debugger" button opens, respectively, the Debugger panel. The "Help" button opens a window with this help. And the About button is a window with brief information about the program and a list of changes (Changelog).
    The "Show debug messages" parameter enables the display of debug messages in the console (via the console.log function).
    The "Disable soft poff" parameter disables support for the software switch present in the processor, which allows the firmware to turn off the device. In MK 87 software shutdown occurs according to a software-implemented timer that counts the time until shutdown. Since the firmware never stops the clock generator of the processor (unlike MK 85), the keyboard polling program is constantly running, which, in the absence of a pressed key, decreases the timer counter (when a key is pressed, the timer is reset). After about 3.5 minutes the timer runs out and the device automatically turns off as a result of a long inactivity.
