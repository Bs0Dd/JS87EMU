Virtual note book


    The appearance of the note book is similar to the real one, only two buttons of the clock module for greater compactness are made inside the metal part of the case. In reality they are located a little further than this part.

Options panel


    Options panel consists of 4 sections and is intended for configuring the emulator. Here you can call special and undocumented functions of the note book (memory reset, show secret word, show firmware author's name), change time on the clock. Pause or restart the emulator, enable or disable vibration when pressing keys (only on mobile devices). It is possible to load RAM from a file and save it. Some debugging and experimental functions are also available, and you can also open a window with a keyboard layout, a help window, the "About" page, and show/hide the Debugger.

Debugger


    Debugger is a debugging complex, more useful for those who want to modify the MK 87 firmware or even create their own. Displays the state of registers (including special ones), allows changing their values. You can control the operation of the device (the "pause" and "restart" keys are duplicated from the options panel), debug the firmware in step-by-step mode, set a breakpoint. There is a RAM editor, with the ability to view the contents in the "bytes" and "words" modes (order from little to big). The complex also includes a lite-disassembler for viewing the operations currently being performed. If desired, any command can be replaced using the lite-assembler. You can switch the display and input modes between hexadecimal and octal.
